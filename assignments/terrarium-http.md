= Exercise: HTTP Backend Request
:icons: font
:source-highlighter: pygments
:pygments-style: borland

:source-language: rust

In this exercise, we will make a simple backend request.

You will learn:

* How to use the `http` crates interface
* How to use terrariums async interface

== Task

1. Create a new Terrarium project
2. Make a HTTP POST request to `https://httpbin.org/post`, forwarding the request body
3. Forward the response body to the client
