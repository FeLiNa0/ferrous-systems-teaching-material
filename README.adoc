# teaching-material

## Prerequisites

You need to have Ruby and bundle installed.

Run

[source]
----
$ bundle --path=.bundle/gems --binstubs=.bundle/.bin
----

Once to install AsciiDoctor

## Building

[source]
----
$ ./rake
$ firefox ./target/index.html
----
